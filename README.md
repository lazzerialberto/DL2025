# INSTALLATION
Clone this repository with `git`:

```shell
git clone https://github.com/lazzerialberto/DL2025.git
```

`Python3` is required. [Miniconda](https://www.anaconda.com/docs/getting-started/miniconda/install) enviroment is adviced.
